## Usage Guidelines

### Buttons
- **Primary Action**: Primary-500 background, Light-50 text
- **Secondary Action**: Outlined with Primary-500, Primary-500 text
- **Destructive**: Error-500 background, Light-50 text

### Cards
- **Surface**: Light-50
- **Shadow**: Dark-300 with 10% opacity
- **Border Radius**: 12px

### Input Fields
- **Fill**: Light-50 (white)
- **Border (Default)**: Light-600
- **Border (Focus)**: Primary-500
- **Border (Error)**: Error-500
- **Border Radius**: 12px

### Status Indicators
- **Success**: Success-500
- **Warning**: Warning-500
- **Error**: Error-500
- **Info**: Secondary-500

## Installation Instructions

### 1. Download Fonts

Visit Google Fonts and download:
- [Poppins](https://fonts.google.com/specimen/Poppins): Regular, Medium, SemiBold, Bold
- [Inter](https://fonts.google.com/specimen/Inter): Regular, Medium, SemiBold, Bold

### 2. Add to Project

Place font files in:
```
housepital/
  assets/
    fonts/
      Poppins-Regular.ttf
      Poppins-Medium.ttf
      Poppins-SemiBold.ttf
      Poppins-Bold.ttf
      Inter-Regular.ttf
      Inter-Medium.ttf
      Inter-SemiBold.ttf
      Inter-Bold.ttf
```

### 3. Uncomment Font Section

In `pubspec.yaml`, uncomment the fonts section (already prepared).

### 4. Run

```bash
flutter pub get
flutter run
```
